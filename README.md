# Comando Zape — Dashboard de Monitoramento WhatsApp

Dashboard centralizado para monitorar grupos WhatsApp via Vigília (agente de monitoramento 24/7).

## Features
- 📊 Visualização de grupos WhatsApp em tempo real
- 💬 Monitoramento de mensagens e atividade
- 👥 Gerenciamento de membros
- 📈 Indicadores KPI centralizados
- ⚙️ Integração com Vigília (agente de monitoramento)
